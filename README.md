# XeeThemeScript

## Installation

  1. Add `xeethemescript` to your list of dependencies in `mix.exs`:

    ```elixir
    def deps do
      [{:xeethemescript, "~> 0.1.0"}]
    end
    ```

  2. Ensure `xeethemescript` is started before your application:

    ```elixir
    def application do
      [applications: [:xeethemescript]]
    end
    ```
